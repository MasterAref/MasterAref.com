سلام

امروز مي خوام درمورد اينکه چه شکلي بروژه اي رو که در کامپيوترمون داريم رو تو Github قرار بديم. در مورد Github بايد بگم که سرويسي هستش که امکانات Git رو که يک Version Controlling system (VCS) هستش رو براي ما بصورت آنلاين فراهم مي کنه.


بجاي اينکه فقط به گفتن اکتفا کنيم و بگيم فلان کد اين کارو مي کنه بهتره که شما هم به همراه من دست به موس و کيبورد شيد. پروژه JizZze رو دانلود کنيد و تو هر مسيري از سيستمتون مي خوايد قرارش بديد. مي خوايم همه فايل هاي موجود توي اين پروژه بجز file1 و file2 رو در دسترس عموم قرار بديم.


خب اولين کار اينه که وارد حساب کاربري تون تو Github بشيد. اگر نيازي به توضيح روش ساخت حساب کاربري بود تو کامنت ها بگيد ولي فعلا از اين موضوع صرفنظر مي کنم. بعد روي علامت بعلاوه، در بالاي سمت راست صفحه کليک کنيد و New Repository رو انتخاب کنيد. در هنگام ساخت repo اين چنيني (سناريوي موجود) نبايد گزينه Initialize this repository with a README رو انتخاب کنيد. بعد از ساخت repo تو قسمت بعدي بايد با استفاده از خط فرمان Git Bash با استفاده از دستور cd به مسير پروژه تون بريد.


در ادامه اگر تنظيمات زير رو هم تنظيم کرده باشيد از قبل کافيست فايل ها رو به Staging Area اضافه و بعد از اون commit کنيد. در نهايت هم عمليات push رو انجام بديد.
